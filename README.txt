CalculatorApp
Описание
CalculatorApp - это простой и функциональный калькулятор с графическим интерфейсом, разработанный на Python с использованием библиотеки Tkinter. Калькулятор поддерживает базовые и расширенные математические операции, а также имеет историю вычислений.
Основные функции:

Базовые операции: сложение (+), вычитание (-), умножение (*), деление (/).
Расширенные операции: остаток от деления (%), возведение в степень (^), квадратный корень (√), синус (sin), косинус (cos), тангенс (tan), натуральный логарифм (ln).
Поддержка скобок для сложных выражений.
История вычислений с прокруткой.
Очистка ввода (кнопка "C"), удаление последнего символа (кнопка "⌫"), очистка истории (кнопка "CH").
Управление с клавиатуры: цифры, операции, Enter для "=", Backspace для "⌫", "c" для очистки ввода, "h" для очистки истории.

Требования

Python 3.6 или выше
Библиотека Tkinter (обычно включена в стандартную установку Python)
Операционная система: Windows, Linux, macOS
Для WSL или серверов: X-сервер (например, VcXsrv для Windows)

Установка

Убедитесь, что Python 3 установлен:
python3 --version


Клонируйте репозиторий с GitHub:
git clone https://github.com/kaarimatajanov/CalculatorApp.git
cd CalculatorApp


Проверьте, что Tkinter доступен:
python3 -m tkinter

Должно открыться тестовое окно.

Если вы используете WSL, настройте X-сервер:

Установите VcXsrv (Windows) или другой X-сервер.

Настройте переменную DISPLAY:
export DISPLAY=$(cat /etc/resolv.conf | grep nameserver | awk '{print $2}'):0.0





Использование

Запустите калькулятор:
python3 calculator.py


Интерфейс:

Введите выражение в поле ввода с помощью кнопок или клавиатуры.
Нажмите "=" или Enter для вычисления.
Используйте "C" (или клавишу "c") для очистки ввода.
Используйте "⌫" (или Backspace) для удаления последнего символа.
Используйте "CH" (или клавишу "h") для очистки истории.
История вычислений отображается в нижнем поле с прокруткой.


Примеры выражений:

2 + 3 * 4 = 14
2^3 = 8
sin(0) = 0
10 % 3 = 1
